Serviço de Usuários:
Microsserviço responsável pelo cadastro, autenticação e gerenciamento de usuários da plataforma de agendamento de tarefas.
Faz parte de um sistema distribuído composto por 4 serviços independentes.

TECNOLOGIAS USADAS
Java + Spring Boot / 
Spring Web /
Spring Security - Autenticação e autorização / 
JWT - Geração e validação de tokens / 
BCrypt / 
PostgreSQL / 
OpenFeign - Integração com a API ViaCEP / 
SpringDoc - Swagger / 
Lombok / 
Gradle / 
Docker e Docker Compose / 
GitHub Actions
